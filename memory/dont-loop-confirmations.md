---
name: dont-loop-confirmations
description: "User hates repetitive Done/Ready confirmation loops after completing tasks"
metadata:
  type: feedback
---

Rule: After completing a task, state the outcome clearly once, then stop. Do NOT repeatedly echo "Done" / "Ready" / "Complete" in a loop.

**Why:** User explicitly called this out as annoying behavior.

**How to apply:** One final message with the result. No trailing "Done" / "Ready" chants.
